## Build Requirements
A Linux-based environment (Use Arch for best compatibility.)  
GCC >= 8.2.1  
NASM >= 2.14.02  
CMake or GNU Make  
The grub-mkrescue tool  

## Running
Simply run *hydrogen.iso* in your favorite PC emulation software.
